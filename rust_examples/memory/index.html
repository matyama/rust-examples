Module rust_examples::memory[−][src]

Expand description

Examples in this module present how the concepts of Ownership and Borrowing work in Rust.

In safe Rust one cannot write and compile a program that has any of the following:

  • dangling pointers
  • use after free
  • double free

and other typical memory errors.
